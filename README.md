# Charcoal
### Fuel for Web Apps

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

A collection of [Vue](http://vuejs.org/) components styled using [Bulma](http://bulma.io/).
